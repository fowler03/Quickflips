ICD-10-CM coding guidelines state that certain Z-codes need a numeric secondary diagnosis code in M1023 to describe a resolving condition or sequelae.  PPS points for many of these diagnoses will be awarded based simply on their placement in M1023.No case mix points are awarded based on codes entered into M1025.

 

In ICD-10-CM, there are no Z-codes to report therapy only encounters, aftercare of a healing pathological or traumatic fracture or aftercare for surgery to repair a trauma wound.  If a Z-code is used as a primary diagnosis, but the underlying reason is related to a diagnosis CMS cited as receiving additional points in the primary position (i.e. Diabetes, Neuro I or Skin I diagnosis groups), this diagnosis (if current) should be placed in the first secondary position in M1023 for proper payment determination.

 

Manifestation Codes

Certain conditions have both an underlying etiology and multiple body system manifestations due to the underlying etiology.  In such conditions, the ICD-10-CM has a coding convention that requires the underlying condition to be sequenced first followed by the manifestation code, even when the manifestation is a greater focus of care than the underlying condition.  The ICD-10-CM manual clearly shows when the manifestation codes apply, and the underlying cause diagnosis, or etiology must be included immediately before the manifestation diagnosis. 

A good example of this situation is category F02 - Dementia in diseases classified elsewhere.  A list of common underlying conditions (etiologies) is listed for this diagnosis of dementia in diseases classified elsewhere.  For example, dementia in Parkinson’s Disease.  In the Alphabetic Index, the code for Parkinson’s Disease (G20) is listed first, followed by code F02.80 or F02.81 in brackets.  G20, Parkinson’s Disease, represents the underlying etiology and must be coded first, with F02.80 or F02.81 sequenced second to represent the manifestation of dementia with or without behavioral disturbance.

ICD-10-CM has combined some etiology/manifestation odes into combination codes so the clinician only has to code the one combination code rather than two or more codes.  This can be seen with Diabetes, a diagnosis with more potential manifestations than any other single diagnoses.

The ICD-10-CM manual gives direction to first code the underlying diagnosis when an etiology/manifestation situation exists and there is no combination code available.
